Baited Server list

Please enter the server id you wish to query in the URL (1-5)

An example would be https://baitedquery.glitch.me/1 for server 1

https://baitedquery.glitch.me/:serverId:/players for currently playing users

https://baitedquery.glitch.me/:serverId:/map for current map on server

https://baitedquery.glitch.me/:serverId:/name for current name of server
